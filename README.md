# Aim

Obolix is a french word-play between `obol` (see bellow) and [Obelix](https://en.wikipedia.org/wiki/Obelix) from the Asterix comic.

Obolix is a fork from the clustervision [obol](https://github.com/clustervision/obol) tool, allowing us to manage users in a really basic LDAP directory structure with CLI.

As Obol was a bit limited in a matter of list of functions, obolix purpose is to extend it to allowing management of homedirs with skeletons (`/etc/skel`), and some other directories (cluster scratchs), but also adding a date function for the expire (`shadowExpire`), etc.

You can check the [TODO](todo.md) list for more informations on what has been done or still a work in progress.

You can see Obolix as Obol under ~~steroids~~ magic potion ! :beaming_face_with_smiling_eyes:

That program has been tested with the help of [podman](README_podman.md).
